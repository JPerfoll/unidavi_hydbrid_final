Backend:

1 - Configurar um servidor apache de sua preferência;
2 - Executar o arquivo "create_database.sql" no servidor apache para criação da base de dados "ionic";
3 - Alterar o arquivo "connect.php" informando as credenciais do seu servidor apache;

App:

1 - Alterar a variável "apiUrl" do arquivo "environment.ts" informando o endereço:porta do seu apache configurado anteriormente;
